# CampusAlert
This is a webpage for a college and school by this School and College can use to maintain the decorum of the college and school in this Student can raise Issue for any thing which is happening in the college like ragging and all this.
CampusAlert ‚Äì Smart Complaint & Alert System for Colleges
 

## Features

### Student Panel
- Secure login/signup system  
- Submit complaints anonymously or with identity  
- Track complaint status in real time  
- Edit/delete own complaints  

###  Admin/Faculty Panel
- Separate secure login for admin/faculty  
- View, filter, and manage all complaints  
- Mark issues as **pending / in-progress / resolved**  
- Send responses or feedback to students
